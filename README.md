AI Applications using Ollama and LangChain

This repository contains a collection of AI-based assignments and mini projects developed using Python, LangChain, Ollama, and Large Language Models (LLMs) such as LLaMA 3 and Phi-3 Mini. The main objective of this project is to demonstrate how modern AI tools can be used to build intelligent applications for text processing, structured data extraction, and real-world automation tasks.

The repository includes multiple assignments that focus on different capabilities of LLMs. The first assignment implements an AI Text Transformer, which accepts a paragraph from the user and performs multiple transformations including generating an academic summary, detecting the tone of the paragraph, and rewriting the content in a formal technical style. This demonstrates how AI can be used for advanced text analysis and rewriting.

The second assignment focuses on structured resume data extraction using LangChain prompts, JSON output parsing, and Pydantic validation. The system reads unstructured resume text and converts it into structured JSON containing details such as name, email, skills, years of experience, and education. This project highlights how LLMs can transform unstructured text into machine-readable data.

The third assignment implements a Product Review Analyzer, where customer reviews are processed to extract structured insights including sentiment, rating, key product features, and improvement suggestions. The system uses Pydantic output parsing to ensure the generated responses follow a strict structured format.

In addition to assignments, the repository contains two mini projects that demonstrate practical real-world AI applications. The first mini project is an AI Job Application Assistant, which analyzes job descriptions, evaluates candidate resumes, suggests improvements, and automatically generates professional cover letters. This tool helps job seekers understand job requirements and improve their application materials.

The second mini project is an AI Medical Report Extractor, which processes medical reports and extracts structured patient information, performs risk assessment, and generates a concise doctor-friendly summary. This project demonstrates how AI can assist in healthcare data processing and information extraction.

All applications run locally using Ollama, which allows large language models to execute offline without requiring external APIs. LangChain is used to design prompt templates, manage LLM pipelines, and enforce structured outputs using Pydantic models. The repository serves as a beginner-friendly demonstration of how to build practical AI systems using modern open-source tools.

Overall, this repository showcases how Generative AI, prompt engineering, structured output parsing, and local LLM execution can be combined to develop intelligent applications for text transformation, data extraction, review analysis, job assistance, and medical report processing.
